一款基于节点树结构的纯前端 Markdown 编辑器。

## 使用方法

```
git clone https://github.com/3253473591/MarkDown-Editor.git
```

双击html即可使用。

## 功能特性

### 节点树管理
- 以树状结构组织所有内容，支持无限层级嵌套
- 拖拽节点自由调整层级与排序
- 支持新建根节点、子节点、同级节点
- 节点支持设置**标签名**与**注释**
- 节点可见性控制：隐藏节点及其子树，隐藏节点不参与导出
- 树结构操作支持最多 50 步撤销 / 重做

### 编辑器
- 左侧编辑、右侧实时预览，分割线可拖动调整比例
- Markdown 工具栏：粗体、斜体、代码、代码块、列表、表格、链接、图片、引用、删除线、上标、下标、分割线
- Tab / Shift+Tab 缩进控制
- 编辑区与预览区滚动同步
- 编辑内容支持最多 100 步撤销 / 重做
- 支持拖入代码文件自动创建节点（js、py、java等 20+ 种格式）

### 导入 / 导出
| 格式 | 导入 | 导出 |
|------|------|------|
| JSON | ✅ | ✅ |
| Markdown | ✅ | ✅ |
| YAML | — | ✅ |
| XML | — | ✅ |
| Word | — | ✅（复制富文本，直接粘贴到 Word） |

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 立即保存 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Ctrl+P` | 全文搜索 |
| `Ctrl+/` | 快捷键一览 |
| `Ctrl+1` | 新建根节点 |
| `Ctrl+2` | 添加子节点 |
| `Ctrl+3` | 添加同级节点 |
| `↑ / ↓` | 树节点导航（非编辑区） |
| `Tab` | 列表缩进 |
| `Shift+Tab` | 列表反缩进 |
| `Insert` | 新建根节点 |
| `Delete` | 删除当前节点（非编辑区） |
| `Escape` | 关闭弹窗 |
